# Hola Mundo Flutter App

Una aplicación simple de "Hola Mundo" creada con Flutter.

## Cómo ejecutar

1. Asegúrate de tener Flutter instalado
2. Ejecuta `flutter pub get` para instalar las dependencias
3. Ejecuta `flutter run` para iniciar la aplicación

## Estructura del proyecto

- `lib/main.dart` - Código principal de la aplicación
- `pubspec.yaml` - Configuración del proyecto y dependencias
